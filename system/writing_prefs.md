# Writing Preferences

- Citation style: Chicago 17th edition, author-date variant
- Bibliography section title: "References" (not "Bibliography")
- Spelling: British English
- Oxford comma: always
- Hedging language: use sparingly — Priya prefers direct claims backed by evidence
- Abbreviations: define on first use, then abbreviate
- Tables: prefer over inline lists for >3 items
- Figures: always include alt-text descriptions
- Reference manager: Zotero (updated to Chicago 17th style)
- Tense: present tense for describing models and methods throughout dissertation (per Morimoto/Webb)
